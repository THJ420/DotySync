Automatically sync products from Dotypos POS (API V2) to WooCommerce.

== Description ==

DotySync for WooCommerce allows you to automatically synchronize your product inventory between your physical store (using Dotypos) and your online store.

**Key Features:**

*   **Seamless Product Sync:** Import product names, prices, and stock levels from Dotypos to WooCommerce.
*   **Real-Time Updates:** Supports Webhooks for instant inventory updates.
*   **Image Handling:** Automatically imports product images associated with Dotypos items.
*   **Category Mapping:** Keeps your online store structure organized.
**Third-Party Service:**

This plugin relies on the **Dotypos Cloud API V2** and **Webhooks** to function. By using this plugin, you are agreeing to send product data to/from the Dotypos Cloud service.

*   [Dotypos Terms of Service](https://www.dotypos.com/terms-and-conditions)
*   [Dotypos Privacy Policy](https://www.dotypos.com/privacy-policy)

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/dotysync-for-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Go to the DotySync settings page and enter your Dotypos Cloud API credentials.
4.  Click "Sync" to start importing your products.

== Frequently Asked Questions ==

= Do I need a Dotypos account? =

Yes, you need an active Dotypos Cloud account to use this plugin.

= Does this sync orders? =

Currently, this plugin specifically synchronizes product data (inventory, prices) from Dotypos to WooCommerce.
